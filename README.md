🏥 Healthcare Data Analysis

📌 Project Overview

This project focuses on analyzing healthcare data to extract meaningful insights about patient health, medical conditions, and hospital costs.
Using data analysis techniques, the goal is to understand patterns in healthcare records and identify factors that influence medical outcomes and expenses.

The analysis helps demonstrate how **data-driven insights can support healthcare decision-making, improve patient care, and optimize resource allocation**.

---

🎯 Objectives

* Explore healthcare datasets and understand patient information.
* Perform **data cleaning and preprocessing**.
* Analyze relationships between **health indicators and medical outcomes**.
* Identify patterns related to **age, BMI, blood sugar levels, and hospital charges**.
* Create visualizations to clearly communicate insights.

---

🗂 Dataset Description

The dataset contains information about patients, including:

* Patient demographics (age, gender)
* Medical conditions
* BMI (Body Mass Index)
* Blood sugar levels
* Smoking habits
* Hospital charges
* Surgery records
* Other healthcare attributes

These variables help analyze **risk factors and healthcare cost trends**.

🔎 Data Analysis Steps

1️⃣ Data Cleaning

* Handling missing values
* Removing duplicates
* Standardizing data formats

2️⃣ Data Transformation

* Categorizing BMI levels
* Classifying blood sugar levels
* Creating new analytical features

3️⃣ Exploratory Data Analysis (EDA)

* Distribution of patient demographics
* Relationship between BMI and diseases
* Blood sugar patterns across age groups
* Hospital charges analysis

4️⃣ Visualization

Charts and graphs were created to understand patterns such as:

* BMI vs health risk
* Age vs hospital charges
* Blood sugar trends
* Disease distribution

---

📊 Key Insights

* Higher BMI levels are often associated with increased health risks.
* Hospital costs tend to increase with age.
* Blood sugar levels play a major role in determining patient health risks.
* Lifestyle factors such as smoking and obesity may influence disease occurrence.

🚀 How to Run This Project

1. Clone the repository

```bash
git clone https://github.com/himnxhu/healthcareAnalysis.git
```

2. Navigate to the project folder

```bash
cd healthcareAnalysis
```

3. Install required libraries

```bash
pip install pandas numpy matplotlib seaborn
```

4. Open the notebook

```bash
jupyter notebook
```

5. Run all cells to reproduce the analysis.
